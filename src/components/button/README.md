| 参数     | 说明                      | 类型    | 可选值                                                    | 默认值  |
| -------- | ------------------------- | ------- | --------------------------------------------------------- | ------- |
| type     | 类型                      | String  | default、primary、warning、danger、success、info、inverse | default |
| size     | 尺寸                      | String  | large、small、default                                     | default |
| disabled | 是否禁用状态              | Boolean | -                                                         | false   |
| block    | 开启后，按钮的长度为 100% | Boolean | -                                                         | false   |
